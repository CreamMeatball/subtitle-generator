## 자막 생성기 v1.0.0 🎬

Whisper(AI 음성 인식)로 **여러 영상의 자막을 한 번에 자동 생성**하는 Windows 앱입니다.
영상을 하나씩 열어 둘 필요 없이, 폴더를 통째로 골라 **백그라운드에서 일괄 처리**합니다.

### ✨ 주요 기능
- 폴더 안 영상들을 골라 **순차/병렬 일괄 자막 생성** (`.srt` / `.smi`)
- **NVIDIA GPU 자동 가속** — 없으면 CPU로 자동 전환
- **번역 자막**: Google · 카카오 · DeepL(API) 또는 로컬 NLLB로 한국어 등으로 번역
- **기존 자막만 번역**: 이미 자막이 있는 영상은 전사 없이 번역만(원본 자동 백업)
- **튜닝 프로필 · 번역 용어집**: 고유명사·전문용어의 인식과 번역 표기를 고정
- 드래그&드롭 추가, 제목 검색, 진행률·처리 시간 표시, 다크/라이트 테마, 자동 업데이트

### 📥 설치 방법
1. 아래 **Assets**에서 `SubtitleGenerator-Setup-1.0.0.exe` 를 받습니다.
2. 실행합니다. 서명되지 않은 앱이라 Windows SmartScreen 경고가 뜨면
   **추가 정보 → 실행** 후 설치하세요(정상입니다).
3. **첫 실행 시** 필요한 구성요소를 자동으로 설치합니다(인터넷 필요, 최초 1회 수 분 소요).
   이후 실행부터는 바로 시작됩니다. 모든 구성요소는 시스템과 분리된 공간에만 설치되어
   기존 환경에 영향을 주지 않습니다.

### 💻 권장 환경
- Windows 10 / 11 (64-bit)
- 인터넷 연결(최초 설정·모델 다운로드 시)
- (선택) NVIDIA GPU + 최신 드라이버 → 자동 가속
