    설정

    tiny=빠름/저정확, large-v3=느림/고정확. 처음 쓰는 모델은 자동 다운로드됩니다.

    병렬 수만큼 모델이 메모리에 올라갑니다. large 계열은 1~2 권장.

    자막 생성이 자꾸 실패하거나 엔진 오류가 나면 눌러 런타임을 깨끗이 재설치하세요. 설정·모델·결과 파일은 그대로 유지됩니다.

    메인 화면 '프로필'에서 선택해 적용합니다. 직접 새 프로필을 추가할 수 있습니다.

    '번역 안 함'이면 원어 자막만 생성합니다. 기본: 한국어.

    ⚠ 로컬 NLLB는 품질이 부족할 수 있어 API 번역(Google·카카오·DeepL) 사용을 권장합니다. API는 아래 키 입력이 필요합니다.

    '원문 => 번역' 형식. 오른쪽을 비우면 그 단어는 번역하지 않고 그대로 둡니다. 메인 화면의 '번역 프로필'에서 선택해 적용합니다.
