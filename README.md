# 자막 생성기 (Subtitle Generator)

<p align="center">
  <img src="https://github.com/user-attachments/assets/2a295575-9801-4098-bc8f-1e7c325833e1" width="200" alt="자막 생성기 로고" />
</p>

**영상 폴더를 통째로 골라, AI 음성 인식(Whisper)으로 자막을 자동 생성하는 Windows 앱입니다.**

영상을 하나씩 열어 두고 기다릴 필요 없이, 폴더 안의 여러 영상을 선택해 **백그라운드에서
순차 또는 병렬로** 자막(`.srt`/`.smi`)을 만들어 줍니다. 만든 자막을 **번역**하거나, 이미
있는 자막만 **번역**할 수도 있습니다.

> 컴퓨터를 잘 몰라도 됩니다. 설치 파일만 실행하면, 필요한 구성요소를 앱이 알아서
> (시스템과 분리된 공간에) 설치합니다.

## 📥 다운로드 & 설치

1. **[Releases](https://github.com/CreamMeatball/subtitle-generator/releases)** 에서
   최신 `SubtitleGenerator-Setup-x.y.z.exe` 를 받습니다.
2. 실행 → SmartScreen 경고가 뜨면 **추가 정보 → 실행** → 설치(서명되지 않은 앱이라 정상입니다).
3. **첫 실행 시** 자동으로 필요한 구성요소를 설치합니다(인터넷 필요, 최초 1회 수 분).
   진행률이 표시되며, 끝나면 바로 사용할 수 있습니다. 이후 실행은 설치 과정 없이 시작됩니다.

설치되는 모든 것은 `%APPDATA%\SubtitleGenerator` 의 **격리된 공간**에만 들어가므로,
PC에 이미 깔린 Python 등과 서로 영향을 주지 않습니다.

---

## ✨ 주요 기능

- **일괄 자막 생성** — 폴더 안 여러 영상을 골라 순차/병렬로 한 번에 처리
- **GPU 자동 가속** — NVIDIA GPU가 있으면 자동 사용, 없으면 CPU로 자동 전환
- **번역 자막** — Google·카카오·DeepL(API) 또는 로컬 NLLB로 한국어 등으로 번역
- **기존 자막만 번역** — 자막이 이미 있는 영상은 전사 없이 번역만(원본 자동 백업)
- **튜닝 프로필 / 번역 용어집** — 고유명사·전문용어의 인식과 번역 표기를 일관되게 고정
- **편의 기능** — 드래그&드롭 추가, 제목 검색, 진행률·처리 시간 표시, 작업 취소/재시도,
  다크/라이트 테마, 자동 업데이트

지원 영상 확장자: mp4, mkv, avi, mov, wmv, flv, webm, ts, m4v, mpg/mpeg, m2ts 등
(오디오 파일 mp3·wav·m4a 등도 가능)

## 🚀 사용법

### 1) 영상 불러오기
- 상단 **📁 폴더 열기** 로 영상이 든 폴더를 선택하거나,
- 창에 영상 파일을 **끌어다 놓기** 하면 목록에 추가됩니다.
- 목록 상단의 **검색창**으로 제목을 필터링할 수 있습니다.

### 2) 영상 선택
- 클릭으로 선택, **드래그**로 여러 개, **Shift**=범위, **Ctrl**=개별 토글, 상단 **전체** 체크.
- 자막이 이미 있는 영상은 목록에 `● srt 있음` 표시가 붙습니다.

### 3) 옵션 지정 (하단 바)
| 항목 | 설명 |
|---|---|
| **모델** | 음성 인식 모델. tiny(빠름) ~ large-v3-turbo(정확·빠름, 기본값) |
| **번역 모델** | 번역 안함(기본) / Google·카카오·DeepL(API) / NLLB(로컬) |
| **원본 언어** | 영상의 말하는 언어. `자동 감지`(기본) 또는 지정 |
| **번역(대상)** | 번역할 목표 언어(기본 한국어). ‘번역 모델’이 ‘번역 안함’이면 미적용 |
| **프로필** | 전사 정확도용 튜닝 프로필(고유명사 주입) |
| **번역 프로필** | 번역 표기 고정용 용어집 |
| **포맷** | 자막 형식(SRT 기본 / SMI) |
| **동시 실행** | 병렬 처리 개수(GPU 메모리에 맞춰 자동 제한) |

### 4) 생성
- **▶ 선택 항목 자막 생성** 클릭 → 오른쪽 **작업 큐**에서 진행률·상태·처리 시간 확인.
- 각 작업은 **취소 / 재시도 / 위치 열기** 가능. 자막은 **영상과 같은 폴더**에 같은 이름으로 저장됩니다.

### 5) 기존 자막만 번역
- 자막(`.srt`/`.smi`)이 이미 있는 영상을 선택하면 **▶ 기존 자막 번역** 버튼이 활성화됩니다.
- 누르면 전사 없이 **번역만** 수행하며, 원본 자막은 `이름.원본.srt` 로 자동 백업합니다(설정에서 변경 가능).

## 🧠 모델 선택

| 모델 | 특징 |
|---|---|
| tiny / base | 가장 빠름, 정확도 낮음(테스트용) |
| small / medium | 균형 |
| large-v3 | 가장 정확, 느림 |
| **large-v3-turbo** | 빠르면서 정확 — **기본값/권장** |

모델은 처음 사용할 때 1회 자동 다운로드됩니다(이후 캐시되어 빠름). GPU가 있으면 큰 모델도
빠르게 처리됩니다.

## 🌐 번역

번역 자막을 만들려면 하단 **번역 모델**을 골라야 합니다(기본은 ‘번역 안함’).

- **API 번역(권장)** — Google 번역 · 카카오 번역 · DeepL. 품질이 가장 좋습니다.
  각 서비스에서 API 키를 발급해 **설정(⚙) → 해당 API 키** 칸에 입력하세요.
  - Google: Google Cloud Translation API 키
  - 카카오: Kakao Developers REST API 키
  - DeepL: DeepL API 키
- **로컬 NLLB** — 인터넷/키 없이 동작(첫 사용 시 모델 1회 다운로드). 다만 품질이
  API보다 떨어질 수 있어, 가능하면 API 사용을 권장합니다.

번역 대상 언어는 하단 **번역(대상)** 에서 고릅니다(기본 한국어). 원본과 대상 언어가
같으면 번역은 자동으로 생략됩니다.

## 🏷️ 프로필 & 용어집

- **튜닝 프로필** (전사 정확도) — 영상 도메인의 고유명사·용어를 **원본 언어로** 적어두면,
  음성 인식이 그 단어들을 더 정확히 받아씁니다. 강의/회의/드라마/뉴스/IT 등 프리셋이 있고,
  **직접 새 프로필을 추가/편집**할 수 있습니다. 설정 → ‘튜닝 프로필 편집’.
- **번역 프로필(용어집)** — 번역 결과에서 고유명사 표기를 고정합니다. `원문 => 번역`
  형식으로 적고(오른쪽을 비우면 원문 유지), 메인 화면 ‘번역 프로필’에서 선택해 적용합니다.

각 작업 카드에는 어떤 모델·프로필·번역 설정으로 처리됐는지 표시되며, **작업마다 독립
적용**됩니다(예: A는 LoL 용어집, B는 다른 설정으로 동시에 처리 가능).

## ⚙️ 설정 (주요 항목)

- **자막 최대 표시 시간(초)** — 한 자막이 너무 오래 떠 있지 않게 제한(기본 7초, 0=무제한).
- **정확한 타임스탬프(단어 단위)** — 시작/끝 시각을 더 정확히(약간 느려짐, 기본 꺼짐).
- **환각 줄이기** — 무음·배경음 구간의 가짜 자막을 억제.
- **기존 자막이 있으면** — 덮어쓰기(기본) / 건너뛰기 / 다르게 저장.
- **기존 자막 번역 시 저장** — 백업 후 저장(기본) / 덮어쓰기.
- **자동 업데이트** — 자동(다운로드 후 안내) / 수동(알림만) + ‘지금 업데이트 확인’.
- **테마** — 우상단 버튼으로 다크/라이트 전환.
- **API 키** — DeepL · Google · 카카오 각각 입력.
- **설정 초기화** — 모든 설정·프로필을 기본값으로(작업 큐·설치 런타임에는 영향 없음).

## 🔄 자동 업데이트

새 버전이 올라오면 앱 시작 시 자동으로 감지합니다. 설정의 ‘자동/수동’에 따라
자동 다운로드 후 **‘지금 재시작’** 으로 적용하거나, 알림만 받습니다. 업데이트는 앱
프로그램만 교체하며, 이미 설치된 구성요소는 그대로 유지됩니다.

## ❓ 자주 묻는 질문 / 문제 해결

- **GPU로 도나요?** NVIDIA GPU + 최신 드라이버면 자동 사용합니다. 안 잡히면 자동으로
  CPU로 동작합니다(느릴 수 있음).
- **자막 타이밍이 어긋나요.** 설정에서 **정확한 타임스탬프(단어 단위)** 를 켜고,
  **자막 최대 표시 시간** 을 조절해 보세요.
- **번역 품질이 아쉬워요.** 로컬 NLLB 대신 **API(Google·카카오·DeepL)** 를 사용하세요.
- **번역이 느려요.** 로컬 NLLB는 CPU에서 느릴 수 있습니다. API 사용 또는 GPU 환경을 권장합니다.
- **앱을 삭제하려면?** Windows 설정 → 앱 → SubtitleGenerator 제거. 데이터까지 완전히
  지우려면 `%APPDATA%\SubtitleGenerator` 폴더도 삭제하세요.
- **설정을 처음 상태로?** 설정창의 **설정 초기화** 버튼.

## 🖥️ 시스템 요구사항
- **OS**: Windows 10 / 11 (64-bit)
- **인터넷**: 최초 설정·모델 다운로드 시 필요
- **저장공간**: 설치 + 구성요소·모델 캐시로 **수 GB 여유** 권장
- **GPU (선택 — 가속용)**: NVIDIA 그래픽카드 + 최신 그래픽 드라이버
  - 권장: **GeForce GTX 10 시리즈 이상** (GTX 16 / RTX 20·30·40 시리즈 등)
  - VRAM(대략): large-v3-turbo·large-v3 ≈ **6GB 이상**, medium ≈ 4~5GB, small 이하 ≈ 2~3GB
  - **CUDA는 따로 설치할 필요 없습니다** — 앱이 첫 실행 시 GPU 런타임을 자동 설치합니다.
  - GPU가 없거나 사양이 부족하면 자동으로 **CPU로 동작**합니다(속도는 느려짐).
